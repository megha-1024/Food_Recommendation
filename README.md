## Food Recommendation System

Data Analyst & Python project using Streamlit, Pandas, and Exploratory Data Processing.

## Project Overview

This project is a Food Recommendation System that recommends recipes based on user-defined nutritional preferences. Users can filter recipes using nutritional values such as calories, protein, carbohydrates, fat, and sodium through an interactive Streamlit web application.

The application processes recipe nutritional information from the Food.com Recipes dataset and provides personalized recipe recommendations based on the selected dietary criteria.

The objective of this project is to demonstrate data preprocessing, nutritional data analysis, and interactive web application development using Python and Streamlit.

## Project Objectives

-Understand and preprocess recipe data.

-Extract nutritional information from the dataset.

-Perform data cleaning and transformation.

-Build an interactive food recommendation system.

-Filter recipes based on nutritional preferences.

-Display recommended recipes in an interactive interface.

-Demonstrate a practical application of Python for recommendation systems.
